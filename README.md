# المساعد الإداري الذكي — كلية الحاسبات والذكاء الاصطناعي

هذا هو المستودع الرسمي لمشروع المساعد الإداري الذكي (AI Agent) الخاص بكلية الحاسبات والذكاء الاصطناعي.

## متطلبات النظام
- Python 3.10+
- Ollama
- (اختياري) Groq API Key

## التثبيت والتشغيل السريع
راجع ملف `smart_admin_assistant_project.md` لمزيد من التفاصيل حول الخطة وسير العمل.

```bash
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt
# تأكد من إضافة GROQ_API_KEY في ملف .env
bash run.sh
```
